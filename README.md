# SparataProject1_TextGame
스파르타 던전 (Text 게임) 만들기


## 🖥️ 프로젝트 소개
### 과제 개요

1. 던전을 떠나기전 마을에서 장비를 구하는 게임을 텍스트로 구현합니다. (C# - Console App)
2. 상점의 아이템 중에서 나만의 장비를 구성하는 부분이 포인트입니다.
3. 장비는 여러개의 데이터가 함께 있는 만큼 객체나 구조체를 활용하는 편이 효율적 입니다.
(이름, 가격, 효과, 설명 등…)
4. 관련된 여러 데이터를 다루는 부분은 배열이 도움이 됩니다.
